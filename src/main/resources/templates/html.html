HTML and CSS Course Profile

Course Overview

The HTML & CSS course is designed for beginners and intermediate learners, covering the essential concepts of web design and development. HTML (Hypertext Markup Language) is the standard language for creating web pages, while CSS (Cascading Style Sheets) controls the layout and styling. Together, they form the foundation of web development, enabling you to build and style interactive web pages

What You'll Learn

  • Understanding HTML structure and elements
  • Creating and organizing content with HTML tags
  • Styling web pages with CSS properties
  • Working with colors, fonts, and layouts
  • Responsive web design using media queries
  • Building navigation menus and interactive buttons
  • Introduction to CSS Flexbox and Grid layouts

Course Duration

8 weeks (4 hours per week)

Prerequisites

No prior web development experience is required, but basic computer skills are helpful
